# Tipos de datos en JavaScript `[]`

```js
typeof ''                // string
typeof 25                // number
typeof false             // boolean
typeof undefined         // undefined
typeof {}                // object
typeof []                // object
typeof null              // object
typeof function(){}      // function
typeof () => {}          // function
typeof 9007199254740991n // bigint
typeof Symbol()          // symbol
```
